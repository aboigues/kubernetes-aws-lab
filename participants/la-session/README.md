# Session de formation Kubernetes

Cette session contient les clés SSH publiques des participants.

## Participants

Chaque participant doit déposer sa clé SSH publique au format `prenom.nom.pub`.

## Format attendu

```
ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAA... [commentaire optionnel]
```

## Exemple

Voir les fichiers `.pub` dans ce répertoire pour des exemples.
